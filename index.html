HTML

  • The head element contains information about the webpage.
    • It is not generally visible to the end-user.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element (which is a box).
  • A padding indicates how much space we want around the content inside an element.
  • A border is the outer extent of the element box. It is outside of the padding and inside of the margin.
  • You can set multiple selectors, seperated by commas, and the following code will effect all of them.

Git

  • git status : checks the branch we're currently on and reports its status.
  • git checkout branchhere : "checks out", or switches to, the specified branch. -b can be used to create a new one.

JavaScript

  • A variable is a container that contains data. 'const' defines a static variable and 'let' defines a dynamic variable.
  • Control flow is the order in which a computer executes code in a script. This can be modified with conditionals.
  • Weakly typed languages allow their variables to implicitly change type according to coercion rules.
  • Dynamically typed languages allow their variables to be created without a defined type. They adopt a type based on what is stored in them.
